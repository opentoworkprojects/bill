# Thermal Print Formats - Quick Reference

## Format Comparison

| Format | Paper Size | Style | Best For | Features |
|--------|-----------|-------|----------|----------|
| **Classic** | 80mm | Traditional | Standard restaurants | Clean, professional, all details |
| **Modern** | 80mm | Contemporary | Cafes, trendy spots | Emojis, borders, visual appeal |
| **Minimal** | 80mm | Simple | Quick service | Essential info only, fast print |
| **Elegant** | 80mm | Formal | Fine dining | Professional, sophisticated |
| **Compact** | 58mm | Space-saving | Food trucks, kiosks | Optimized for small printers |
| **Detailed** | 80mm | Comprehensive | Formal billing | Complete invoice format |

## Printer Settings

### For 80mm Printers
- Paper Width: 80mm (3.15")
- Margin: 5mm each side
- Font Size: 12px
- Formats: All except Compact

### For 58mm Printers
- Paper Width: 58mm (2.28")
- Margin: 3mm each side
- Font Size: 10px
- Format: Compact only

## Common Customizations

### Header Customization
- Restaurant name (always shown)
- Tagline (optional)
- Logo (where supported)
- Address (recommended)
- Contact info (phone, email, website)

### Legal Information
- GSTIN (for GST compliance)
- FSSAI (food safety license)
- Business registration number

### Footer Customization
- Thank you message
- Website URL
- Social media handles
- Promotional text
- Return policy

## Tips for Best Results

### Design Tips
1. Keep restaurant name short (< 30 chars)
2. Use clear, simple tagline
3. Ensure phone number is correct
4. Add website for online orders
5. Keep footer message friendly

### Technical Tips
1. Test print after setup
2. Check paper alignment
3. Verify all details visible
4. Ensure text not cut off
5. Adjust format if needed

### Maintenance Tips
1. Keep printer clean
2. Use quality thermal paper
3. Replace paper regularly
4. Check printer settings
5. Update drivers if needed

## Mobile Printing

### Android/iOS
1. Open billing page
2. Complete payment
3. Click print icon
4. Preview opens
5. Use "Share" to send to printer app
6. Or email receipt to customer

### Bluetooth Printers
1. Pair printer with device
2. Install printer app
3. Configure in app settings
4. Print from preview window
5. Or use app's print function

## Troubleshooting Quick Fixes
